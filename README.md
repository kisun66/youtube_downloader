# youtube_downloader
Flask, python, html/css, jquery

pytube를 이용한 플라스크 기반 유튜브 동영상 다운로드 사이트입니다.

주소를 입력하면 Downloads 폴더에 동영상이 "년-월-일-시-분-초.mp4"라는 제목으로 저장이 됩니다.

그리고 사이트에는 동영상 제목과 썸네일이 띄워지고 다운로드 버튼이 새로 생깁니다.

다운로드 버튼을 누르면 다운로드가 시작됩니다.
